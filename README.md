# safetyexploreR

## Create [Rho's](https://github.com/RhoInc) interactive safety explorer suite from R.  
### SafetyexploreR provides an R interface for the 6 safety explorer JavaScript widgets.  This set of tools can be used individually or together, allowing a user to explorer adverse events and labs data in real time.

### Safety explorer widgets include:
- AE Explorer
- AE Timelines
- Safety Histogram
- Safety Outlier Explorer
- Safety Results Over Time
- Safety Shift Plot

### safetyexploreR is an R package built using the [HTML Widgets](http://www.htmlwidgets.org/) framework.
